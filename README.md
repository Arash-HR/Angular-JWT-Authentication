# Requirements
- Angular CLI: 13.2.1
- Node: 16.13.2
- Package Manager: npm 8.1.2

# Installation

## Development server
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. 

## Running json-server
Run `npm run server` for run backend api with json-server.

# Login
Home route is protected and you can access it if you are logged in, use `admin@admin.com` and `admin` as username and password.
